Why does my RemoteEvent fire on the client but nothing happens on the server?
Classic issue. You need OnServerEvent on the server, not OnClientEvent. Also make sure both scripts reference the same RemoteEvent in ReplicatedStorage.
Should I use FireServer or FireAllClients?
FireServer goes client to server. FireAllClients goes server to all clients. For a shop purchase: client fires FireServer, server validates, then fires FireAllClients to update everyone's UI.
