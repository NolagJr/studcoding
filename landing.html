Why does my RemoteEvent fire on client but nothing happens on the server?
You need OnServerEvent on the server side, not OnClientEvent. Also double-check both scripts are referencing the exact same RemoteEvent object in ReplicatedStorage.
Should I use FireServer or FireAllClients?
FireServer is client → server. FireAllClients is server → every client. For a shop: client fires FireServer, server validates the purchase, then fires FireAllClients to update all UIs.
