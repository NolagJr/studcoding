Guide
How It Works
Get set up and coding smarter in minutes.
1
Create your account
Sign up at StudCoding. Your account tracks XP, quiz scores, AI memory, and chat history.
2
Install the Roblox plugin
Search "StudCoding" in the Roblox Creator Store or use the link in Plugin Setup.
3
Get your Plugin Key
Generate a key in Plugin Setup, copy it, and paste it into the plugin inside Roblox Studio.
4
Code smarter
Use the Studio plugin to fix bugs and write scripts, or use this dashboard to study and get quizzed.
