// How to Use
Get Started with Stud
Everything you need to know to go from zero to building.
1
Get your SC Plugin Key
Go to Get Plugin in the sidebar and copy your SC-XXXXX-XXXXX key. You'll need this to activate the plugin in Roblox Studio.
2
Install the Studio Plugin
Copy plugin.lua into your Roblox plugins folder. Keep the server running while you use Roblox Studio.
C:\Users\[you]\AppData\Local\Roblox\Plugins\
3
Start the Server
Open your Rocode folder in a terminal and run the server. Keep it running while you use Roblox Studio.
python server.py
4
Activate & Connect AI
In the plugin, paste your SC key and verify with the email code. Then go to Connect AI here to link your API key for more power.
5
Fix and Write Scripts
Paste a broken script or describe a new one. Stud fixes it, reviews it 4 times, then injects it directly into Studio.

Connect Stud to Roblox Studio
Three steps. Takes under a minute.
1
Install the Plugin
Get the official Stud plugin from the Roblox Creator Store. Click the button and it installs directly into Studio.
🧩 Open in Roblox Studio
2
Copy Your Key
This is your unique access key. Keep it private.
Click generate to get your key
Plugin status: Not connected
3
Paste into the Stud Plugin
Open Roblox Studio. Click the Stud button in the plugins toolbar. Paste your key and hit Connect. You're in.
// Connect AI
AI Settings
Connect your own API key for more control and power. If you don't add one, Stud uses the shared server key.
🔌 Provider Settings
Your key is stored encrypted and never shared.
ℹ About AI Keys
Without your own key, Stud uses the server's shared key which may have rate limits.

With your own key, you get full speed, higher limits, and your usage stays separate from other users.
