Why does RemoteEvent fire on client but nothing happens server-side?
You're using OnClientEvent on the server — that only fires on clients. Switch to OnServerEvent and make sure both scripts reference the exact same RemoteEvent in ReplicatedStorage.
Should I use FireServer or FireAllClients?
FireServer is client → server. FireAllClients is server → every client. For a shop: client fires FireServer, server validates, then FireAllClients updates all UIs.
