Guide
How It Works
Get set up in minutes.
1
Create your account
Sign up at StudCoding.app. Your account tracks your XP, quiz scores, and AI memory.
2
Install the Roblox Plugin
Search "StudCoding" in the Roblox Creator Store and install it into Roblox Studio.
3
Get your Plugin Key
Go to Plugin Setup in this dashboard, generate your key, and paste it into the plugin inside Studio.
4
Start coding smarter
Use the plugin to fix bugs, write scripts, and review code — or use this dashboard to study with lessons and quizzes.
About
StudCoding
StudCoding is an AI-powered coding assistant built specifically for Roblox developers. Whether you're just starting out or working on a complex game, it helps you write better Lua faster — fixing bugs, teaching concepts, and reviewing your scripts in real time.
